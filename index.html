UI Comment Tool — Reviewer 使用示例

点击下方"确认下载"按钮后，浏览器会下载一个 comment.json 示例文件。

如何使用：

  1. 在打开的 Reviewer 页面中设置一个昵称
  2. 点击页面中的 Import 按钮
  3. 选择刚刚下载的 comment.json 文件导入
  4. 即可在原型图上看到示例评论标注

建议使用 Chrome 内核浏览器以获得最佳体验。
