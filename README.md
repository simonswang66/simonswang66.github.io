# Simons Work Suites

个人日常工具套件，托管于 GitHub Pages。

## 工具列表

### 工作工具

- **[UI Comment Tool](ui-comment-tool/builder.html)** — UI 原型图评审协作工具，上传原型图、配置页面、导出分发包
- **[AI API Demo](ai-api-demo/index.html)** — AI API 调用演示，支持 OpenAI / DeepSeek，Cloudflare Worker 安全代理
- **[交易对手方打标工具](counterparty-tool/index.html)** — 交易对手方列表批量打标与浮动面板管理
- **[UI Design Samples](ui-design-samples/index.html)** — 13 种 UI 设计风格组件预览
- **[知识库](knowledge-base/index.html)** — 日常学习与知识总结，审计业务、产品设计等领域学习笔记

### 宝宝乐园

- **[字母学习乐园](baby-paradise/字母学习乐园.html)** — 键盘字母/数字答题，看字母按对应键，答对星星动画
- **[魔法敲击乐园](baby-paradise/魔法敲击乐园.html)** — 自由敲击，字母蹦跳 + 音效 + 动画，按住不放连续播放
- **[趣味画板](baby-paradise/趣味画板.html)** — 三种绘画模式：自由涂鸦（悬浮笔/连续画）、英文数字书写（描字母，填满自动跳）、你画我猜（即将上线）

## 使用方式

纯静态 HTML，浏览器直接打开即用。`git push` 自动部署到 GitHub Pages。
